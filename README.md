# EMİR HAYAL'in DİKKATİNE

1. Projede web sayfaları **/templates** klasörüne konacak bu klasördeki dosyalar görüntü sağlayacak.

2. Tailwind'in **input.css** dosyası **/public/css/** içindedir **output.css** de orda oluşacak şekilde ayarlanacak.

3. Websitesinin tasarımındaki tüm fotoğraflar **/public/assets/images** içinde olacak.

4. PHP dosyaları ve backend ile alakalı kodlar **/src/** klasörü içindedir.

### 1. Projeyi değiştirmeye başlamadan önce githubdan maain branchi çek
Bu projenin temel halidir değişiklikler bu kodun üzerine yapılır.

Kodun main branchi ile birleşecektir
